# Очередь студентов к 3 компьютерам

## Описание
Программа моделирует очередь студентов к 3 компьютерам. Одновременно работают только 3 студента, остальные ждут.

## Запуск
dotnet run

## Пример работы
Сколько студентов? 5
Студент 1 ждёт...
Студент 1 сел за комп
Студент 2 ждёт...
Студент 2 сел за комп
Студент 3 ждёт...
Студент 3 сел за комп
Студент 4 ждёт...
Студент 5 ждёт...
Студент 1 закончил
Студент 4 сел за комп
Все студенты закончили!

## Технологии
C#, .NET, SemaphoreSlim

## Теория
1. TCP-клиент подключается, сервер ждёт
2. DLL - библиотека с кодом
3. Race condition - потоки мешают друг другу
